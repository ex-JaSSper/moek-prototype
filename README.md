# Прототип портала МОЭК с AI-помощником

Демонстрационный прототип: локальная копия портала ekp.moek.ru
со встроенным чат-ботом на базе RAG.

## Возможности

- Навигация по разделам портала
- Чат-помощник (n8n + RAG по базе знаний МОЭК)
- Выбор языковой модели
- Ссылки на источники с автоматической подсветкой ответа на странице
- Голосовой ввод

Прототип создан в демонстрационных целях.
